# PHP Socket Ping Pong

Обмен сообщениями между сервером и клиентом.

## Установка
* `composer install`
* `cp .env.example .env`
* настроить переменные .env по желанию

## Использование
* `php server.php`
* `php client.php`

## Что можно добавить
* поддержка аргументов из командной строки
* поддержка нескольких клиентов
